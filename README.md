# TOF monster timetable

[https://hoshiyuki-tamako.github.io/tofmt/](https://hoshiyuki-tamako.github.io/tofmt/)

For managing farming tof boss items

## requirement

- nodejs 18

## setup / running for development

```sh
npm i
npm run dev
```

### Build

```sh
npm run build
```

## main framework / library use

- vuejs 3
- pinia
- element-plus
- peerjs
- class-transformer
- class-validator
- dayjs
- linq
